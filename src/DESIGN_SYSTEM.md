# Scotland Luxury Estates Design System

## Vision

Scotland Luxury Estates should feel like an architectural magazine, not a SaaS application.

Think:

- JamesEdition
- Financial Times HTSI
- Monocle
- Architectural Digest
- Apple

Luxury through typography, spacing and photography.

Not through decoration.

---

# Design Principles

## 1. Sharp edges

No rounded cards.

No rounded buttons.

No rounded drawers.

Everything is square.

---

## 2. Borders over shadows

Prefer

border

instead of

shadow

Only use subtle shadows where absolutely necessary.

---

## 3. White space

Always add more breathing room than feels necessary.

Luxury websites never feel crowded.

---

## 4. Typography first

Typography should create hierarchy.

Avoid decorative UI.

Headings should be allowed to dominate.

---

## 5. Photography first

Images should be the hero.

UI should support photography, never compete with it.

---
